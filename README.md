# Selene

一个以 MoonTV v100 版本 / Helios 为后端的客户端，支持 iOS、Android、MacOS 和 Windows，并针对各端体验进行优化。

## 致谢

特别感谢原作者 **[MoonTechLab](https://github.com/MoonTechLab)** 提供的优秀底层架构与灵感。本项目在其基础上进行了功能扩展与多端体验优化。

## 特性

- **多端支持**: 全面适配 iOS, Android, MacOS, Windows。
- **播放优化**: 基于 `media_kit` 深度优化的播放体验。
- **自动更新**: 内置版本检测与更新功能。

## 构建

项目根目录下提供了 `build.sh` 脚本，支持一键并行构建各平台版本。
